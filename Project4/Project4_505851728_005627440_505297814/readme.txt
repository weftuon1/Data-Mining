Python3 setup via Conda environment
Jupyter Notebook

Required Python Packages (installed in Conda environment):
pandas
numpy
nltk
scikit-learn
scikit-surprise
catboost
matplotlib
json
datetime, time
pytz
csv
re
seaborn
cudf
cuml

Steps to run:
1. Make sure your conda environment and jupyter notebook are appropriately set up.
2. Make sure all the Python packages are installed.
3. Activate your conda environment and activate Jupyter Notebook.
4. Place the dataset in the specific directory mentioned in the notebook.
5. Install cudf and cuml for part2_twitter data.
6. Open the notebook and sequentially run all the sections.
