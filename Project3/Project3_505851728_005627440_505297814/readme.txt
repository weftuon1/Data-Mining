Python3 setup via Conda environment
Jupyter Notebook

Required Python Packages (installed in Conda environment):
pandas
numpy
nltk
scikit-learn
scikit-surprise
matplotlib


Steps to run:
1. Make sure your conda environment and jupyter notebook are appropriately set up.
2. Make sure all the Python packages are installed.
3. Activate your conda environment and activate Jupyter Notebook.
4. Place "Synthetic_Movie_Lens" folder in the same directory where the notebook is located.
5. Open the notebook and sequentially run all the sections.
